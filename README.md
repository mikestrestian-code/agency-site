# Vantage Logic — Agency Website

## ⚠️ CRITICAL: How to upload to GitHub Pages

### The correct way to upload:

1. Go to your GitHub repo: https://github.com/mikestrestian-code/agency-site
2. DELETE all existing files in the repo first
3. Click "Add file" → "Upload files"  
4. EXTRACT the zip on your computer first, then drag these items into GitHub:
   - index.html
   - css/ folder
   - js/ folder
   - pages/ folder
   - assets/ folder
5. Commit changes → wait 60 seconds → visit your site

### The repo root must look like this:
```
index.html          ← right here at root level
css/main.css
js/main.js
pages/about.html
pages/work.html
pages/careers.html
pages/contact.html
assets/images/logo.png
assets/images/favicon.png
```

### Do NOT upload the zip directly or the agency-site folder itself.
